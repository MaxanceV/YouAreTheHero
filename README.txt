YOU ARE THE HERO

=================================================================

Lancement du Jeu
Prérequis : 
	Java Runtime Environment (JRE) version 11 ou supérieure.
	Pour Lancer le Jeu
	Télécharger le fichier JAR : Téléchargez le fichier YouAreTheHero.jar depuis l'emplacement fourni.

Exécuter le fichier JAR depuis Windows, macOS et Linux :
	Ouvrez un terminal et tapez la commande suivante :
	java -jar VousEtesLeHeros.jar

==================================================================

Premières Étapes :
Au lancement du jeu, vous aurez le choix entre faire une nouvelle parti ou chargé une parti précédente.
Lancer une nouvelle parti vous invitera à créer un personnage. Son nom vous sera utile pour retrouver la sauvegarde. 
Chargé une sauvegarde vous permettra de choisir parmis les sauvegardes effectuées.

Commencer l'aventure : 
Une fois un personnage sélectionné, le jeu commence par une introduction à l'histoire. Lisez attentivement les informations présentées car elles pourraient vous donner des indices sur les choix à faire.

Faire des choix : 
À chaque point de décision, vous aurez plusieurs options. Ces choix peuvent mener à différentes branches de l'histoire et influencer le résultat final. Sélectionnez l'option qui vous semble la plus appropriée ou intéressante pour progresser.

Quitter le Jeu :
Pour quitter le jeu, naviguez jusqu'à l'option de menu qui permet de quitter, ou fermez simplement la fenêtre du jeu. Votre progression est sauvegardée automatiquement à chaque étape importante, vous permettant de reprendre là où vous vous étiez arrêté.
